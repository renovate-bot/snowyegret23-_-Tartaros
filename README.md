# Tartaros

yt-dlp 기반 다운로드 GUI (Windows exe 기준 사용법)

## 설치/실행

1) Releases에서 exe 다운로드  
2) 원하는 폴더에 exe를 놓고 실행

처음 실행하면 같은 폴더에 아래 파일이 생성됩니다.
- `settings.ini` : 모든 설정 저장
- `list.csv` : 다운로드 기록
- `cookies.txt` : 설정창에 붙여넣은 cookies.txt 내용을 저장한 파일
- `.thumbnails/` : 썸네일 캐시

## 기본 사용법

1) 메인 화면 아래 입력창에 YouTube URL 입력  
2) `다운로드` 클릭 (또는 Ctrl+Enter)  
3) 항목 진행 상태는 목록에서 확인

## 설정 창

메뉴 → 설정

### 1) 언어 설정
앱 언어를 즉시 변경합니다. (영어/한국어/日本語/简体中文/繁體中文)

### 2) 일반 설정
- **저장 폴더**: 다운로드 저장 위치 (상대 경로면 exe 폴더 기준)
- **파일명 템플릿**: 기본 파일명 포맷  
  예) `[%(uploader)s] %(title)s (%(id)s).%(ext)s`
- **플레이리스트**: 플레이리스트 저장 경로/파일명 포맷  
  예) `[Playlist] %(playlist_title)s/%(playlist_index)03d. %(title)s.%(ext)s`

템플릿 변수 예시: `%(title)s`, `%(uploader)s`, `%(id)s`, `%(ext)s`, `%(playlist_title)s`, `%(playlist_index)s`

### 3) 다운로드 설정
- **출력 포맷**: mp4/mkv/webm 또는 오디오 전용(mp3/m4a/opus/flac)
- **화질**: best/2160p/1440p/1080p/720p/480p/360p/오디오만
- **음질**: best/320k/256k/192k/128k/96k
- **동시 조각 수**: 분할 다운로드 동시 수
- **재시도 횟수**: 다운로드 실패 시 재시도 횟수
- **다운로드 후 파일 검증**: Best 화질 선택 시 무결성 확인

### 4) 자막/메타
자막
- **자막 다운로드**
- **자동 생성 자막 포함**
- **자막 언어**: 쉼표 구분 (예: `ko,en,ja`)
- **자막을 영상에 임베드**

메타데이터
- **썸네일 임베드**
- **썸네일 별도 저장**
- **메타데이터 추가**

### 5) 인증
브라우저 쿠키
- **브라우저 쿠키 사용**: 브라우저에 로그인된 쿠키 사용
- **브라우저**: Chrome/Firefox/Edge/Opera/Brave/Vivaldi/Safari

성인 콘텐츠
- **연령 제한 영상 다운로드 허용**: 브라우저 쿠키 사용 시 필요

쿠키 (텍스트)
- **cookies.txt 내용**: 브라우저에서 추출한 Netscape 형식 cookies.txt 내용을 그대로 붙여넣기  
  (형식 참고: `www.youtube.com_cookies.txt`)

### 6) SponsorBlock
- **SponsorBlock 사용**
- **제거할 구간**: 쉼표 구분 (예: `sponsor,intro,outro`)
- **챕터로 표시**: 제거 대신 챕터로 표시할 카테고리

## 메인 화면 옵션

하단 옵션
- **클립보드**: 클립보드에서 URL 자동 감지
- **정렬 버튼**: 최신순/오래된순 토글

다운로드 항목 우클릭 메뉴
- **영상 주소 복사**
- **다시 다운로드**
- **오류 발생한 항목 모두 다시 다운로드**
- **정상적으로 다운로드된 항목 모두 제거**
- **삭제**
- **잠금**

## 주의사항

- `settings.ini`는 exe와 같은 폴더에 생성됩니다.
- `cookies.txt`는 설정에 붙여넣은 내용을 파일로 만들어 yt‑dlp에 전달합니다.
- 네트워크 환경/로그인 상태에 따라 다운로드 동작이 달라질 수 있습니다.
